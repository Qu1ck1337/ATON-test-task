This is a test task for ATON company

# Stack
- **Backend:** Python (FastAPI), SQlite
- **Frontend:** HTML, CSS, JS, React

# How to run
**Run frontend:**
`npm start` on '/frontend' folder

**Run Backend:**
`python main.py` on '/backend' folder
